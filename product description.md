## Product concept

A premium “color mixing” play-and-learn game for kids (and design-minded adults) built with Toca Boca–grade craft: tactile interactions, no junk UI, slow delight, high replayability. The core fantasy is **being a color alchemist**—mixing pigments, predicting outcomes, collecting discoveries—across multiple modes that all reinforce the same skill: **seeing color as a system** (hue, value, saturation, temperature) rather than a list of names.

## Positioning

A single app with a clear umbrella theme (**Mix Lab**) and three pillars:

1. **Solve** (target-matching puzzles)
2. **Predict** (reverse puzzles / mental model building)
3. **Collect** (diary + recipes + personal palette)

No ads. No dopamine traps. Short sessions that stack into mastery.

## Target audience

* Primary: kids ~5–10 (parent-approved, classroom-friendly)
* Secondary: older kids / adults who enjoy visual puzzles, design, illustration, makeup, painting, photo editing

## Core loop

1. Pick a mode (Solve / Predict / Collect)
2. Do a short “mix session” (15–90 seconds)
3. Earn **knowledge rewards** (new pigments, new palette trays, new tools, new pages)
4. Build your **Color Diary** (save recipes and favorites)
5. Return with new constraints (more colors, smaller deltas, fewer moves)

## Design principles (Toca-quality bar)

* **Tactile mixing:** thick paint, viscosity, believable blending, smears, palette knife, droppers, sponge, brush swirls.
* **Instant feedback:** you always see the mix evolving; the UI never blocks the color.
* **Constraint-based fun:** “limited pigments,” “limited scoops,” “no undo,” “must use 3 colors,” etc.
* **Calm polish:** soft sound design, subtle haptics, generous motion, zero clutter.
* **Two mixing models:** default **pigment/subtractive** (paint-like), optional **light/additive** (RGB) as a separate “lab room.”

# Modes / Games inside

## Game A — “Match the Target” (Solve)

You’re shown a target swatch. You must create it using the available pigments and tools.

### Level 1: Pick the pigments

* A small set of pigments is offered.
* You choose which ones are needed (no quantities yet).
* The learning: recognition of dominant hue + obvious modifiers (white/black/complements).

### Level 2: Pick pigments + amounts

* Same as above, but you drag “scoops” or “drops” into the bowl.
* The learning: ratio thinking (more white vs a tint; more blue vs a cooler shift).

### Level 3: 3+ pigments required

* Targets are designed so two-color solutions fail.
* The learning: neutralizing with complements, warming/cooling, desaturation control.

### Level 4: “Sparse palette / spectrum primaries”

* You have a constrained “student palette,” or a spectrum-like primary set (separated primaries).
* Targets require planning; the palette expands over time.
* The learning: pigment limitation strategies and path-to-color logic.

### “Hard v2” inspiration (TryColors / MixToMatch)

* You see candidate pigments and must reach the target in the fewest moves.
* Optional constraints: limited total volume, no undo, limited mixing steps (A+B then +C).

**Scoring** is non-toxic: accuracy band + elegance badges (“1-step match,” “perfect ratio,” “used minimal pigments”).

---

## Game B — “Predict the Result” (Predict)

Inverted puzzles. The app shows the input pigments (and their amounts). You must pick the resulting color.

### Easy: obvious outcomes

* Red + white → pink
* Blue + yellow → green (in pigment model)

### Medium: ratios and tints/shades

* “Two drops red + one drop white” vs “one drop red + two drops white”
* Introduces value shifts and saturation.

### Hard: complements, neutralization, temperature

* “Orange + blue” produces muted browns/greys depending on ratio.
* Challenges become about *where the result lands* in a tight selection.

### Very hard: multi-step mixing

* A+B creates intermediate; then mix with C.
* Player predicts the final without doing it physically.

This mode builds the internal color model fast, and it makes Game A feel easier (because prediction is training).

---

## Game C — “Color Diary” (Collect / Create)

A personal notebook where mixing becomes expressive, not just correct.

### Free Mix Pages

* Infinite canvas pages where you can mix freely.
* Save swatches, name them, and pin “recipes” (pigments + ratios + steps).
* Optional “stamp” tool to create gradients, palettes, mood boards.

### Recipe Cards

* Each saved color becomes a card: swatch, formula, and a tiny “story” (where you used it).
* Export/share is optional; the core is collection pride.

### Daily Prompts (optional)

* “Make the color of sea glass.”
* “Make a shadow color.”
* “Make a warm grey.”
  Prompts reinforce concepts without feeling like school.

---

## Game D — “Find the Twin” (Discrimination)

(Your “pick 1 among similar colors” game—this is a killer mode if done with polish.)

### Level 1: obvious deltas

Pick the exact match among 6 similar blues.

### Level 2: smaller deltas

Differences become subtle in value or saturation.

### Level 3: many swatches

More candidates, smaller differences, time pressure optional.

### Level 4: contextual perception

Swatches appear on different backgrounds or with lighting shifts to teach relativity (simultaneous contrast).

This mode makes the app feel “premium” because it’s perceptual training disguised as a clean game.

# Progression and retention (without junk)

* Unlock new **pigments** (limited edition pigments: “ultramarine,” “quinacridone rose,” “sap green”) and tools (knife, dropper, blender).
* Unlock new **challenge packs** (Sunset, Skin tones, Metals, Food, Nature, Architecture).
* Unlock new **lab rooms** (Pigment Lab, Light Lab, Mystery Lab).
* A simple “collection wall” shows your discovered colors as tiles—your personal universe.

# Monetization / packaging

* Premium paid app or premium + single unlock (no subscriptions).
* Optional “Creator Packs” as one-time purchases (new pigments + challenge packs + diary stickers).

# Brand and tone

Playful, minimal, design-forward. The app is a **toy-grade creative tool** wrapped in a **puzzle-grade learning loop**. The promise: you’ll leave with a sharper eye and a private library of colors you actually made.
